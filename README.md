## 🌟 Overview

**QR for You** is an advanced, standalone, single-page web application designed to generate and fully customize QR codes. Built with React and Tailwind CSS, it offers a beautifully minimalist UI paired with a highly powerful customization engine. 

Whether you need a simple QR code for a website or a heavily branded QR code with custom gradients, logos, eye shapes, and frames, **QR for You** delivers a frictionless, premium experience.

## ✨ Features

- **🎨 Advanced Customization Studio:**
  - **Color Control:** Solid colors, linear gradients, and radial gradients.
  - **Shape Design:** Customize the internal dots (Square, Rounded, Circle, Diamond, Classy, Modern Pixel, etc.).
  - **Eye Design:** Customize the corner eye shapes (Square, Rounded, Leaf, Modern, Minimal).
- **🖼️ Logo Integration:** Upload custom logos or use built-in presets (Facebook, Instagram, LinkedIn, YouTube, Twitter) and adjust their size.
- **🖼️ Custom Frames:** Add frames around your QR code with customizable call-to-action text and colors (Classic, Speech Bubble, Minimal Border, Business Card).
- **🤖 AI Color Suggestions:** Instantly generate beautiful color palettes using intelligent presets (Sunset, Ocean Breeze, Green Forest, Royal Lavender, Dark Gold).
- **📱 Responsive & Fast:** Fully optimized for all devices, delivering an App-like experience on mobile.
- **📥 Multiple Export Formats:** Download your custom QR code in high-quality PNG, SVG, or PDF formats.
- **🌍 Multi-language Support:** Ready for localization.

## 🚀 Quick Start

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/mna80455-debug/qr-for-you.git
   cd qr-for-you
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 🛠️ Tech Stack

- **Framework:** React 18 + TypeScript
- **Bundler:** Vite
- **Styling:** Tailwind CSS + Framer Motion
- **QR Code Engine:** `qrcode-generator`
- **PDF Generation:** `jspdf`
- **Icons:** Lucide React
- **Hosting:** Vercel
